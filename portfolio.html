Past Experiences

Description

In September 2022, I began my journey into the world of programming with C++. During this time, I developed a solid understanding of the language and applied my skills to create a project titled "Guess The Number." This project helped me gain practical experience and deepen my understanding of programming fundamentals. Moving forward to February 2023, I delved into Object-Oriented Programming (OOP) with Java. This phase of my learning was marked by the development of a "Train Booking System" project. This project not only reinforced my knowledge of Java but also taught me how to design and implement complex systems using OOP principles. By August 2023, I had progressed to studying Data Structures, a critical component of computer science. My project during this period was the "Phone Dialer" application, which utilized the stack data structure. This project enhanced my ability to efficiently manage and manipulate data, a skill that is essential for any software developer. In February 2024, I expanded my skillset to include Database Management Systems (DBMS) with a focus on SQL. To solidify my learning, I developed a "Hotel Management System" project. This experience provided me with hands-on knowledge of database design, querying, and management, which are crucial for backend development. Most recently, in June 2024, I ventured into web development with HTML, CSS, and JavaScript. This led to the creation of my "Portfolio Web" project, a personal website that showcases my skills and projects. This project allowed me to integrate my technical knowledge with creative design, resulting in a professional and visually appealing online presence.
